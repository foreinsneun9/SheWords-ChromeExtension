# SheWords

一个 Chrome 浏览器扩展程序（也可直接插入Edge浏览器），自动识别并替换网页中的性别偏见词汇，让语言更加中性和包容。

## ✨ 主要功能

### 🔄 智能词汇替换
- **内置词库**：预设 30+ 常见性别偏见词汇的替换规则
- **自动替换**：实时监测网页内容，自动替换匹配的词汇
- **高亮显示**：替换后的词汇会以浅红色背景高亮，方便识别
- **大小写智能匹配**：
  - 搜索时不区分大小写（`police officer` 可以匹配 `Police officer`）
  - 替换时保留原词的大小写格式（`Policeman` → `Police officer`，`POLICEMAN` → `POLICE OFFICER`）

### 📝 自定义词库
- **添加自定义词汇**：支持单个添加或批量添加
- **持久化存储**：自定义词汇可选择持久保存或仅在当前会话有效
- **导入/导出**：支持 JSON 格式的词库文件导入导出，方便分享和备份
- **独立记录管理**：删除添加记录不影响实际替换效果

### ⚙️ 灵活控制
- **一键开关**：随时启用或禁用替换功能
- **动态更新**：添加新词后立即生效，无需刷新页面

## 📦 安装方法

### 方式一：从源码安装（开发者模式）

1. 下载或克隆本项目到本地
2. 打开 Chrome 浏览器，访问 `chrome://extensions/`（Edge：edge://extensions/）
3. 开启右上角的「开发者模式」
4. 点击「加载已解压的扩展程序」
5. 选择 SheWords 项目文件夹
6. 安装完成！扩展图标会出现在浏览器工具栏

## 🎯 使用指南

### 基础使用

1. **启用扩展**：点击工具栏中的 SheWords 图标，确保开关处于「开启中」状态
2. **浏览网页**：正常浏览任何网页，扩展会自动识别并替换性别偏见词汇
3. **查看替换**：被替换的词汇会以粉色背景高亮显示

### 添加自定义词汇

#### 单个添加
1. 点击扩展图标打开弹窗
2. 在「原词」和「替换词」输入框中填写内容
3. 勾选「同步到外部词库（持久化）」可永久保存，不勾选则仅在当前会话有效
4. 点击「添加」按钮

#### 批量添加
1. 在「批量添加」文本框中输入内容，支持两种格式：
   - JSON 格式：`{"原词1":"替换词1", "原词2":"替换词2"}`
   - 行分隔格式：每行一对，如 `原词1=>替换词1`
2. 选择是否持久化
3. 点击「批量添加」按钮

### 导入/导出词库

- **导出**：点击「导出外部词库文件」按钮，下载 JSON 格式的词库文件
- **导入**：点击「导入外部词库文件」按钮，选择之前导出的 JSON 文件

### 管理添加记录

- 在「自定义词库添加记录」列表中，点击词条右侧的「×」按钮可删除该记录
- **注意**：删除记录不会影响已添加词汇的替换效果，词汇仍会继续生效

## 🛠️ 技术实现

- **Manifest V3**：使用最新的 Chrome 扩展规范
- **内容脚本**：通过 TreeWalker 遍历 DOM 文本节点进行替换
- **MutationObserver**：监听页面动态内容变化，实时替换新增内容
- **存储分离**：替换词库与显示记录分离存储，互不影响

## 🤝 贡献指南

欢迎提交 Issue 和 Pull Request！

如果你有更多性别偏见词汇的替换建议，可以：
1. 提交 Issue 说明词汇对
2. 在 `external_dictionary` 文件夹新增你的词库文件(命名建议使用你的词库主题，如德语替换词、网文替换词等)并提交 PR
3. 通过扩展的导出功能分享你的自定义词库
4. 将你希望添加的词汇对及对SheWords的建议填入表格：https://docs.qq.com/sheet/DVFdYSXhrWXpVdkNC?nlc=1&tab=BB08J2
